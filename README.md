# AI Student Success Prediction Dashboard

This project predicts whether a university student will:

- Graduate
- Stay Enrolled
- Drop Out

## Technologies Used
- Python
- Scikit-learn / XGBoost
- Streamlit
- Google Gemini API

## Run the Project

Activate virtual environment

source venv/bin/activate

Set Gemini API key

export GEMINI_API_KEY="YOUR_API_KEY"

Run the dashboard

streamlit run app/streamlit_app.py
